# Azraeel

I build systems for simulation, game design, and structured worldbuilding.

Most of my work sits at the intersection of strategy games, military simulation, procedural tools, and persistent fictional worlds. I focus on projects that need more than surface-level features: dynamic behavior, readable data structures, long-term maintainability, and tools that help complex communities actually run.

## Current Focus

* Developing tools and infrastructure for **AG-GS / Avant-Garde**, including live world-state tracking, national statistics, wiki systems, admin workflows, and simulation support.
* Building and improving **Arma 3** mission systems, AI behavior, logistics, faction configuration, and dynamic battlefield mechanics.
* Working on better documentation standards for large, evolving projects.
